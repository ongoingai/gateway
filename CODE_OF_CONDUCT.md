# Code of Conduct

## Our Standard

This project is committed to a respectful, professional, and inclusive community.

Examples of expected behavior:

- Be respectful and constructive in technical discussions.
- Assume good intent, ask clarifying questions, and focus on facts.
- Accept feedback gracefully and collaborate on solutions.

Examples of unacceptable behavior:

- Harassment, threats, or discriminatory language
- Personal attacks, trolling, or bad-faith disruption
- Sharing private information without consent

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

If you experience or witness unacceptable behavior, contact maintainers through GitHub project channels. For security-related concerns, use `SECURITY.md` reporting guidance.
